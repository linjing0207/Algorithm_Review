1. N个结点的二叉树，有多少种形态？  
   N=1, f(1)=1   
   N=2, f(2)=f(1)+f(1)  
   N=3, f(3)=f(2)+f(1)*f(1)+f(2)  
   N=4, f(4)=f(3)+f(2)*f(1)+f(2)*f(1)+f(3)  
   ...  
   N=n, f(n)=f(0)f(n-1)+f(1)f(n-2)+...+f(n-2)f(1)+f(n-1)f(0) <f(0)=1>  
   这就是卡特兰数(Catalan)。  
   前几个数为1,1,2,5,14,42,132。  
   
   java code:  
```import java.util.Scanner;
   public class Main {

       public static void main(String[] args) {
           Scanner in = new Scanner(System.in);
           int n = in.nextInt()；
           System.out.print(catalan(n));
       }

       public static int catalan(int n){
           int[] h = new int[n +  1];
           h[0] = h[1] = 1;
           for(int i = 2; i <= n; i++){
               h[i] = 0;
               for(int j = 0; j < i; j++){
                   h[i] += h[j] * h[i - 1 - j];
               }
           }
           return h[n];
       }
   }```
   

2. N个结点的二叉树，限制高度为m，有多少种形态？  
   java code:  
 
 
   `import java.util.Scanner;
   public class Main {

       public static void main(String[] args) {
           Scanner in = new Scanner(System.in);
           int n = in.nextInt();
           int m = in.nextInt();
           System.out.print(catalan(n, m));
       }

       public static int catalan(int n, int m){
           int[] h = new int[n +  1];
           h[0] = h[1] = 1;
           for(int i = 2; i <= n; i++){
               h[i] = 0;
               for(int j = 0; j < i; j++){
                   if(j >= m || (i-1-j) >= m) continue;
                   h[i] += h[j] * h[i - 1 - j];
               }
           }
           return h[n];
       }
   }`
